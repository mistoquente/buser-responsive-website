Como funciona

1. Preparou É preparar o seu Tang. É bolar um plano para melhorar o planeta, e juntar os amigos para fazer. 2. Bebeu É tomar um gole de seu Tang preferido, um gole que dá força pra você e a turma toda. Só assim para encarar os desafios. 3. Faz Você preparou, você bebeu, você faz. É uma mistura de Tang, amigos e idéias simples. Idéias para fazer um planeta melhor.
